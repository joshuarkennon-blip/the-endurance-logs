Project Overview // April 2026

THE ENDURANCE LOGS

An immersive digital archive dedicated to the filmography of Christopher Nolan.

↗ Live Site ↗ GitHub ↗ Vercel

What It Is

A high-end fan site styled as an analog-future spacecraft console — the Endurance from Interstellar. Users load film "discs" via drag-and-drop, triggering a boot sequence before revealing richly written dossiers on each film's philosophical themes and technical craft. Built for atmosphere first: CRT scanlines, film grain, LED indicators, and a brushed-metal panel aesthetic throughout.

Components & Functions

Console
components/Console.js
Root layout component. Owns all state (loaded film, loading phase, zoom, drag). Renders the outer frame — header bar, instrument panel, status strip — and coordinates DiscShelf, LogScreen, and the drop zone.
DiscShelf
components/DiscShelf.js
Left-side archive panel. Renders each film as a draggable disc card with color-coded spine, pixel thumbnail, film code, and active/loaded state indicator. Fires drag events up to Console.
LogScreen
components/LogScreen.js
Main display area. Three internal states: idle (no disc), loading (boot sequence), and active (dossier view). Houses both sub-components below.
LoadingSequence
components/LogScreen.js
Animated boot screen that plays on disc drop. Streams filesystem-swap log lines one by one, advances a progress bar, then fires onComplete to transition into the dossier view.
DossierView
components/LogScreen.js
Renders the full film dossier — title, tagline, transmission metadata, thematic sections — with staggered section-by-section reveal on mount. Includes EJECT button to unload the disc.
Drag & Drop Handler
components/Console.js
Native HTML5 drag API wired across Console. handleDragStart sets dragging state, handleDrop reads the filmId from dataTransfer, looks up the film object, and triggers the load sequence + zoom.
CRT / Grain Overlay
styles/globals.css
::before and ::after pseudo-elements on .crt create a repeating scanline grid and an animated sweep line. A separate .grain div uses an SVG feTurbulence filter animated across random offsets to simulate analog film grain.
